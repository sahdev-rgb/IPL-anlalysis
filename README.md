 # IPL Analysis Dashboard

## Overview

This project is an IPL Analysis Dashboard created in Excel to analyze Indian Premier League data from 2008 to 2024. The dashboard provides insights into team performance, toss decisions, venue statistics, award winners, and title-winning teams.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Conditional Formatting

## Dashboard Features

* Season-wise filtering (2008–2024)
* Winner and Runner-up details
* Player of the Match and Player of the Series
* Team wins based on Bat First and Field First
* Toss decision analysis
* Top venues analysis
* Top 10 Player of the Match winners
* IPL title winners comparison

## Key Insights

* Analyze the impact of toss decisions on match results.
* Compare team performance while batting first and chasing.
* Identify the most successful IPL teams and players.
* Understand venue-wise winning trends.

## Dataset

The dataset contains IPL match information including season, teams, venue, toss winner, match winner, and player awards.
